Detailed Description

State storage for 24dB/octave bandpass filter in AudioEquationParser.

BP24State implements a two-pole IIR bandpass filter (cascaded 12dB stages) for use in audio equations. The filter provides a steeper slope for more selective bandpass filtering with narrower bandwidth.
