Detailed Description

State storage for 24dB/octave lowpass filter in AudioEquationParser.

LP24State implements a two-pole IIR lowpass filter (cascaded 12dB stages) for use in audio equations. The filter provides a steeper 24dB/octave slope for more aggressive lowpass filtering.
